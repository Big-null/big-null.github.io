# 項目細節規劃：
以下是一些規劃建議：

1. 先決定 `項目主題`
2. 再考慮所需的 `功能`（我們的PWA需要什麼）
3. 想想 `風格` 和 `主題`
4. 確定我們需要哪些 `頁面` 以及由誰來負責
5. 開設一個 Github 儲存庫並邀請所有團隊成員。
6. 先創建這些空白的 `頁面` / `文件夾` / `檔案` 並進行 `git push`。
7. 其他團隊成員 `git pull` 檔案並開始工作。"
